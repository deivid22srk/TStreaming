# File Streaming Bot

**File Streaming Bot** é uma solução completa para hospedar e gerenciar o bot de streaming de arquivos diretamente do seu dispositivo Android. Baseado em tecnologia de emulação de terminal moderna, este aplicativo automatiza toda a configuração do ambiente Alpine Linux, Python e as dependências necessárias para o funcionamento do bot.

## 🚀 Funcionalidades
- **Setup Automatizado:** Configuração completa do Alpine Linux, Python e Git com um único clique.
- **Interface Intuitiva:** Gerencie o servidor através de uma interface dedicada, sem precisar digitar comandos complexos.
- **Logs em Tempo Real:** Acompanhe o progresso da instalação e a execução do bot em um mini terminal integrado.
- **Gerenciamento de Configurações:** Exporte e importe suas configurações do bot (API ID, Tokens, etc) facilmente via JSON.
- **Terminal Integrado:** Acesso total ao terminal Alpine Linux quando necessário.
